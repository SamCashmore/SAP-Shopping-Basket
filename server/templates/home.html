Dyson Web Store using SAP Customer Order Sourcing

Welcome to an example Dyson e-commerce web store built using Flask.

Calls are made to the SAP Customer Order Sourcing microservice, where users can check stock quantities of Dyson Products and reserve products close to their location.

To use this service:

  • Visit the products page

  • Select items (only V11 for now)

  • Select desired quantity - up to a maximum of the number available

  • Add items to basket

  • Reserve items by entering postcode

Visit SAP-Shopping-Basketfor source code.
